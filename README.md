## About Advent of Code ‚ÑπÔ∏è

[Advent of Code](https://adventofcode.com/) is an online programming competition created by Eric Wastl. It consists of a series of programming puzzles released daily during the month of December. Participants solve the puzzles using their programming skills and can discuss solutions on the Advent of Code website.

## Folder Structure üìÇ

- `day1/`: Contains the solutions for Day 1.
- `day2/`: Contains the solutions for Day 2.
- ...
- `day25/`: Contains the solutions for Day 25.

Each day's folder may include the following:

- `input.txt`: The input data for the puzzle.
- `part1.py`: The solution for Part 1 of the puzzle.
- `part2.py`: The solution for Part 2 of the puzzle.
- ...

Feel free to explore each day's folder for the respective solutions.
